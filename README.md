# 국어 기말 빈칸 퀴즈 📚

고등 국어(문학) **갈래별·시대별 핵심 정리**를 빈칸 채우기로 연습하는 웹앱입니다.
고전 문학과 현대 문학의 **작품명·작가명·핵심 내용** 347문항을 담았습니다.

## ▶️ 바로 쓰기
- `index.html` 을 더블클릭하면 브라우저에서 바로 열립니다. (설치·인터넷 불필요)
- 휴대폰에서 쓰려면 이 파일 하나만 전송해도 되고, 아래 GitHub Pages로 링크를 만들면 어디서나 접속할 수 있습니다.

## ✨ 기능
- **두 가지 학습 방식**
  - ✍️ 타이핑 연습 — 정답을 직접 입력 (띄어쓰기·문장부호는 무시하고 채점)
  - 👆 빈칸 클릭 — 타이핑 없이 빈칸을 눌러 정답 확인 후 자가 채점
- 범위 선택: 전체 / 고전·운문 / 고전·산문 / 현대·운문 / 현대·산문
- 유형 선택: 전체 섞기 / 작가명 / 작품명 / 핵심 내용
- 정답 시 **칭찬 메시지 + 연속 정답(콤보) 표시**, 결과 화면·**오답 노트**·틀린 문제만 다시 풀기
- 컴퓨터·아이폰 모두 지원하는 반응형 디자인

## 🌐 GitHub Pages로 링크 만들기 (선택)
1. 저장소 **Settings → Pages** 이동
2. **Source**: `Deploy from a branch`, **Branch**: `main` / `/(root)` 선택 후 저장
3. 1~2분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 접속 (파일명이 `index.html`이라 주소만으로 바로 열립니다)

## 📂 구성
- `index.html` — 앱 본체 (이 파일 하나면 전부 동작)
- `README.md` — 설명
